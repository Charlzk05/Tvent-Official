## Tvent
A simple Twitter BOT made in node js for Everyone to use.

### Packges
- ``dotenv``
- ``ejs``
- ``express``
- ``twitter-api-v2``

### Installation
- ``git clone https://github.com/Charlzk05/Tvent-Official.git``
- replace the app keys, secret and access token, secret
- do ``npm run devstart``
